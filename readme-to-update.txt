Update guide:

​Requirement for this Writebot 4.4.0 version, 

1. Current Application Version 4.3.0

2. There you will get a zip folder named ‘updateVersion4.4.0’. Upload that to the root directory on your server in which your previous version is running. Right Click on the zip file. Unzip that "updateVersion4.4.0.zip" file by selecting “Extract here”.

3. Now go to the home page of your website(wwww.yourdomain.com) and click on ‘Update Now’.

4. Update will be completed!
